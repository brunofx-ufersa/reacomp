Gráficos

Para usar a biblioteca Plots, certifique-se de que está instalada.

using Plots
gr()

gr é o backend padrão.

Gráfico de linha simples

x = 1:10
y = x.^2

p1 = plot(x, y, 
            title="Linha", 
            xlabel="x", 
            ylabel="y", 
            legend=false, size=(600, 400))

ATRIBUTOS PRINCIPAIS:

Atributo Significado Exemplo
title Título do gráfico title="Título"
xlabel, ylabel Rótulos dos eixos xlabel="eixo_x"
legend Posição da legenda legend=:topright
color Cor da linha color=:red
alpha Transparência (0 a 1) alpha=0.5
linewidth Espessura da linha linewidth=2
linestyle Estilo linestyle=:dash
markersize Tamanho do marcador markersize=8
size Tamanho da figura (largura, altura) size=(600, 400)

Gráfico de dispersão

x_scatter = randn(100)
y_scatter = randn(100)

p2 = scatter(x_scatter, y_scatter, 
                title="Dispersão", 
                xlabel="x", 
                ylabel="y", 
                legend=false)
x_erro = 1:5
y_erro = [2, 4, 6, 8, 10]
erro = [0.5, 0.8, 0.6, 0.9, 0.7]

p_erro = scatter(x_erro, y_erro, yerror=erro, 
                  title="Com Barras de Erro",
                  xlabel="x", ylabel="y", 
                  legend=false, markersize=8)

Múltiplos gráficos (subgrafico)

x = 0:0.1:2π

p1 = plot(x, sin.(x), title="sin(x)", label="sin(x)")
p2 = plot(x, cos.(x), title="cos(x)", label="cos(x)")
p3 = plot(x, tan.(x), title="tan(x)", label="tan(x)", ylims=(-5, 5))
p4 = plot(x, exp.(-x), title="e^(-x)", label="e^(-x)")

p_layout = plot(p1, p2, p3, p4, layout=(2,2), size=(800, 600))
  • ylims(), xlims(): Define limites dos eixos

Histograma

dados = randn(1000)

p1 = histogram(dados, title="Barra")
p2 = scatterhist(dados, title="Scatter")
p3 = stephist(dados, title="Step")

plot(p1, p2, p3, layout=(1, 3), legend=false)

Gráfico com múltiplas linhas

x = 0:0.05:10
y1 = sin.(x)
y2 = cos.(x)
y3 = sin.(x) .* exp.(-x./10)

p1 = plot(x, y1, label="sin(x)", linewidth=2)
p2 = plot!(x, y2, label="cos(x)", linewidth=2)
p3 = plot!(x, y3, label="sin(x)·e^(-x/10)", linewidth=2, linestyle=:dash)

p_multi = plot(p3, 
                title="Múltiplas Funções", 
                xlabel="x", ylabel="y", 
                legend=:topright, size=(700, 500))
  • plot!(): Adiciona elementos ao plot anterior

Simular dados de temperatura ao longo do tempo

using Random
Random.seed!(42)

dias = 1:30
temperatura = 20 .+ 5 .* sin.(2π .* dias ./ 7) .+ randn(30) .* 0.5
umidade = 60 .+ 20 .* sin.(2π .* dias ./ 10) .+ randn(30) .* 2

# Plot com dois eixos Y
p_temp = plot(dias, temperatura, 
              title="Temperatura vs Umidade",
              xlabel="Dias", ylabel="Temperatura (°C)",
              label="Temperatura", color=:red, 
              linewidth=2, legend=:topleft)

p_duplo = twinx(p_temp)
p_salvar = plot!(p_duplo, dias, umidade, 
      label="Umidade", color=:blue, 
      linewidth=2, ylabel="Umidade (%)", 
      legend=:topright)
  • twinx(): Cria segundo eixo Y
savefig(p_salvar, "grafico.png")
savefig(p_salvar, "grafico.pdf")
savefig(p_salvar, "grafico.svg")
