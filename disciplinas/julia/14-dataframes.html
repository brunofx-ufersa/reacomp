DataFrames

Certifique-se de que as bibliotecas using XLSX e DataFrames estejam instaladas. Nesta etapa, preparamos o ambiente Julia e garantimos que as dependências necessárias para leitura de arquivos Excel e manipulação de dados estejam disponíveis.

Aqui carregamos as bibliotecas que serão utilizadas ao longo do tutorial.

using XLSX, DataFrames

Criando um dataset

Agora, criamos um DataFrame a partir de uma planilha específica do arquivo. Isso permite trabalhar com os dados de forma estruturada. Em seguida, visualizamos as primeiras linhas para entender o conteúdo.

df = DataFrame(XLSX.readtable("WorldCups.xlsx", "WorldCups"))
first(df, 3) # Seleciona as três primeiras linhas

Selecionando colunas

Nesta etapa, removemos colunas que não são necessárias para a análise. Isso ajuda a manter o dataset mais limpo e objetivo.

select!(df, Not([:QualifiedTeams, :Attendance]))
names(df) # mostra os nomes das colunas

using CSV
CSV.write("WorldCups.csv", df)
